## Description
This project is a simple e-commerce application inspired by Meesho. 
It allows users to browse products, view product details, and place orders.

- HTML
- CSS
- JavaScript
- MongoDB
- Node.js

## Features
- User login and registration
- Product browsing
- Add to cart
- Order placement
